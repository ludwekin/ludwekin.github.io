```markdown
### 学习 CMake 的方法

#### 1. 了解 CMake 基础概念
- **CMake 是什么**：跨平台构建系统生成工具，用来生成 Makefile 或 IDE 工程文件。  
- **用途**：管理 C/C++ 项目编译、链接依赖库、跨平台构建。  
- **核心概念**：
  - `CMakeLists.txt`：项目配置文件  
  - `add_executable()`：添加可执行文件  
  - `add_library()`：添加库  
  - `target_link_libraries()`：链接库  
  - `find_package()`：查找系统或第三方库

#### 2. 学习步骤
1. **基础教程**
   - 官方文档：[https://cmake.org/cmake/help/latest/](https://cmake.org/cmake/help/latest/)  
   - 简单项目：创建单文件 C++ 项目，用 CMake 编译。
2. **逐步扩展**
   - 多文件项目  
   - 使用库（静态库/动态库）  
   - 自定义编译选项  
   - 条件编译（不同平台或配置）
3. **学习模块化**
   - 使用 `add_subdirectory()` 管理子模块  
   - 使用 `install()` 安装目标
4. **学习进阶**
   - 使用 `FetchContent` 或 `ExternalProject` 引入第三方库  
   - 自定义 CMake 函数和宏  
   - CTest 集成单元测试  
   - 配合 CI/CD 自动构建（GitHub Actions, GitLab CI）

#### 3. 实践练习
- 用 CMake 编译已有开源项目（如 gRPC、RocksDB）  
- 尝试修改 `CMakeLists.txt` 添加新源文件或依赖  
- 将小项目迁移到 CMake 管理
```
