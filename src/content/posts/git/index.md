## 常用命令，以及解释

1. git push origin --delete feature/stable_fly.

    这个命令删除远端分支。

2. 看当前项目（当前目录）对应的是哪个 Git 仓库 / 远端仓库。

    git remote -v.

3. 在当前分支下开创一个新分支然后继续开发。

    git switch -c feat/fractal_aruco_tag.

    git switch -c fix/improve_fractal_detect.
